# 红米K80测光表 (RedmiK80 Light Meter)

专为红米K80优化的专业测光应用，基于光影猎人800传感器的精确照度测量。

## 功能特点

### 1. 多种测光模式
- **环境光传感器模式**: 使用手机环境光传感器进行入射光测光
- **相机测光模式**: 使用后置相机进行反射光测光

### 2. 专业曝光计算
- 照度测量 (Lux)
- 曝光值计算 (EV)
- 快门速度计算
- ISO/光圈/EV补偿调节
- 曝光组合建议

### 3. 校准功能
- 与专业测光表对比校准
- 补偿手机传感器与专业设备的差异
- 支持重置校准

### 4. 针对红米K80优化
- 主摄传感器: 光影猎人800 (豪威OV50E定制版)
- 光圈: f/1.6
- 等效焦距: 24mm
- 传感器尺寸: 1/1.55英寸

## 技术实现

### 使用的技术
- **语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: MVVM + StateFlow
- **最低SDK**: Android 7.0 (API 26)
- **目标SDK**: Android 14 (API 34)

### 核心组件
- `LightSensorManager`: 环境光传感器数据采集
- `ExposureCalculator`: 曝光参数计算工具类
- `LightMeterViewModel`: 业务逻辑和状态管理
- Jetpack Compose UI组件

## 使用说明

### 基本测量
1. 打开应用后，应用会自动读取环境光传感器的数据
2. 上方显示当前的照度(Lux)和曝光值(EV)
3. 下方显示在不同ISO和光圈组合下的建议快门速度

### 调整曝光参数
1. 使用ISO滑块设置感光度 (25-12800)
2. 使用光圈滑块设置光圈值 (f/1.0-f/22)
3. 使用EV补偿滑块进行曝光微调

### 校准步骤
1. 点击右上角的调谐图标打开校准设置
2. 在相同光照条件下，使用专业测光表测量照度
3. 输入专业设备测得的参考照度值
4. 点击"完成校准"，系统会自动计算并应用偏移量

### 曝光组合建议
应用会根据当前EV值，自动计算并显示常用的曝光组合:
- ISO 100 + f/2.8
- ISO 100 + f/4.0
- ISO 100 + f/5.6
- ISO 200 + f/2.8
- ISO 200 + f/4.0
- ISO 400 + f/2.8
- ISO 400 + f/4.0
- ISO 800 + f/2.8

## 常见问题

### Q: 为什么测量值与专业测光表有差异？
A: 手机光传感器与专业测光表的响应曲线不同，建议使用校准功能进行补偿。

### Q: 支持哪些手机？
A: 本应用专为红米K80优化，但理论上支持所有具有环境光传感器的Android设备。

### Q: 如何提高测量精度？
A:
- 确保传感器窗口干净
- 避免手指遮挡传感器
- 在稳定光照条件下测量
- 定期进行校准
